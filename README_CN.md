## 它做什么？

给你的 AI 助手配备一位资深设计总监的审美眼光。根据 Apple HIG、Material Design 3 和「年轻奢华」美学体系审查移动端 UI，返回**可直接落地的代码级建议**。

每次响应提供 **三层方案**：

| | 方案 A: 稳妥 | 方案 B: 均衡 ⭐ | 方案 C: 前卫 |
|---|---|---|---|
| **策略** | 平台原生组件 | 80% 平台 + 20% 品牌个性 | 重度动效 + 玻璃拟态 |
| **开发成本** | 低 | 中 | 高 |
| **惊艳程度** | ★☆☆ | ★★☆ | ★★★ |

建议精确到具体数值 — Spring 阻尼、字距百分比、WCAG 对比度、圆角半径、阴影层级。
